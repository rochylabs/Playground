# Goethe-Zertifikat A1 — Prüfungsvorbereitung

Interactive exam prep app for the Goethe-Zertifikat A1 (Start Deutsch 1), built with Next.js.

Covers all 4 exam sections: Lesen, Hören, Schreiben, Sprechen — with 4 rotating exam sets, pre-generated German TTS audio, and pattern tips guided by mascot Max 🐶.

## Getting Started

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) to use the app locally.
